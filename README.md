# 使用说明

首先在本仓库中创建分支，分支名为你的github id，如```lwzhenglittle```。

在自己分支的```README.md```中更新自己的进度，commit并push到仓库中，每次push都需要开一个PR请求将自己的分支合并到主分支```main```中，主分支每天进行合并请求的处理。

禁止任何成员直接操作主分支，同时在编辑过程中也要注意不要修改他人的进度。
